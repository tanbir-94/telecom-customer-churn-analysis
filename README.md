# Telecom Customer Churn Analysis & Executive Dashboard

## Project Overview

Customer churn is one of the biggest challenges faced by telecom companies as it directly impacts revenue growth and customer lifetime value.

This end-to-end data analytics project analyzes customer churn behavior using Python-based statistical analysis and an executive-level Power BI dashboard to identify churn drivers, revenue risk segments, and customer retention opportunities.

The project demonstrates a complete analytics workflow followed by professional data analysts — from raw data cleaning and exploratory data analysis (EDA) to statistical validation and executive business storytelling.

---

## Business Problem

High customer churn leads to significant revenue loss for telecom companies.

The objective of this project is to:

- Identify churn patterns.
- Understand customer segments at risk.
- Analyze service usage behavior.
- Support retention strategy using data-driven insights.

---

## Project Workflow

### 1. Data Cleaning & Preparation (Python)

Performed using Python libraries:

- Pandas
- NumPy

Cleaning Steps:

- Removed missing values.
- Corrected data types.
- Handled duplicates.
- Created derived analytical columns.

---

### 2. Exploratory Data Analysis (EDA)

Key analysis performed:

- Customer demographics analysis.
- Contract type comparison.
- Payment method behavior.
- Monthly vs Total charges comparison.
- Senior citizen churn trend.
- Tenure vs churn relationship.

---

### 3. Statistical Analysis

Statistical validation performed using hypothesis testing:

- T-Test
- Chi-Square Test
- Effect Size Analysis

Purpose:

To validate whether observed churn differences between customer groups are statistically significant.

---

### 4. Executive Power BI Dashboard

Designed as a senior-level executive dashboard focused on business decision making.

Dashboard Highlights:

- KPI Cards (Total Customers, Churn Rate, Revenue Impact)
- Customer Segmentation Analysis
- Contract Type Risk Analysis
- Payment Method Risk Comparison
- Service Usage Impact Analysis
- Interactive Filters and Navigation

---

## Key Business Insights

- Month-to-month contracts show the highest churn risk.
- Customers using electronic check payment methods churn more frequently.
- Short tenure customers are more likely to leave early.
- Higher monthly charges correlate with churn probability.

These insights help telecom companies prioritize retention strategies.
